# Знайди Помилку

Знайди помилку в коді і виправ її!

### Завдання 1

#### Код із помилкою:

```javascript
let age = "8;
console.log(age);
```

#### Пояснення:

Помилка через те, що рядок не закритий лапками.

#### Правильний код:

```javascript
let age = "8";
console.log(age);
```

---

### Завдання 2

#### Код із помилкою:

```javascript
let favoriteColor = blue;
console.log("Мій улюблений колір: " + favoriteColor);
```

#### Пояснення:

Текст значення змінної має бути в лапках.

#### Правильний код:

```javascript
let favoriteColor = "blue";
console.log("Мій улюблений колір: " + favoriteColor);
```
